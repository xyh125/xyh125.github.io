# 浏览器API访问限制

由于VuePress应用程序在生成静态版本时是在Node.js中服务器渲染的，因此任何Vue用法都必须符合[通用代码要求]（https://ssr.vuejs.org/en/universal.html）。 简而言之，请确保仅在“ beforeMount”或“ mounted”挂钩中访问浏览器/ DOM API。
